Welcome to Apps@Stern

Apps@Stern provides NYU Stern's students and faculty with virtual access to academically relevant software applications.

To access additional applications NYU offers, please navigate to the NYU Virtual Computer Lab (VCL).

Use of Apps@Stern and the Virtual Computer Lab (VCL) are for academic purposes only.

For more information, please see the Apps@Stern service documentation.
